# Bookie — Premium Infant Clothing Store

An e-commerce storefront for Bookie, a premium infant clothing brand.

## Tech Stack

- **Framework**: React + TypeScript (Vite)
- **Styling**: Tailwind CSS + shadcn/ui
- **State**: Zustand (cart), React Query (data fetching)
- **Backend**: Shopify Storefront API, Supabase (auth)
- **Hosting**: Vercel

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```
